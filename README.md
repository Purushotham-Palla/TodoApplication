# ToDoList Application

ToDoList Application is a simple web-based task management application.

## Features

- Add tasks with due dates
- Mark tasks as completed
- Edit task details and Project Titles
- Delete tasks 

## Tech Stack

- Java
- JDBC
- JSP
- Servlets
- CSS
- MySQL database
